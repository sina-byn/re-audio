# useVisualizerFrame

---

The `useVisualizerFrame` hook is a wrapper around the [`useVisualizer`](/docs/audio-visualizers/useVisualizer) hook that integrates the `getFrequencyData()` function with `requestAnimationFrame()`.
It returns a `frame`, which is a `number[]` representing the frequency data as a React state.

This frame updates in real-time as the audio plays, allowing you to render your visualizer without any additional concerns about data management or updates.

This hook accepts an optional `dataPoints` parameter, which defaults to `64` if not specified. This value determines the length of the `frame`.
